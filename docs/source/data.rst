Datasets
========

All tomobank data sets are distributed in the native file format or in data exchange :cite:`decarlo:02`. To 
perform a basic reconstruction using tomoPy  :cite:`Gursoy:14a` use: ::

    $ tomopy recon --file-name tomo_0001.h5 --rotation-axis 1024.0
