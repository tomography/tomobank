XFusion
-------

XFusion is a deep learning-based spatiotemporal fusion method for high-fidelity ultra-high-speed x-ray radiography.
It reconstructs high-quality x-ray image sequences by combining the high spatial resolution of a high-speed camera
with the high temporal resolution of an ultra-high-speed camera sequence.

* Code: https://github.com/xray-imaging/XFusion
* Documentation: https://xrayfusion.readthedocs.io/

If you use XFusion, we kindly request that you cite :cite:`Tang:25`.
